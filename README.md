What is Pseudocode and why do programmers use it?
Pseudocode is a simple way of describing an algorithm using plain English instead of a programming language. It helps programmers plan, organize, and understand the logic of a program before writing code.
Five Rules of Pseudocode
Begin with START.
End with END.
Write one instruction per line.
Use clear and simple English.
Use indentation to show structure in decisions and loops.
